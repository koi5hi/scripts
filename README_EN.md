# 📖 Scripts
Random scripts I wrote for fun, updated regularly. If you find them useful, please give a ⭐!

## Repository Overview
- **ddns-scripts**
  Simple DDNS script, compatible with Cloudflare.
- **earnapp-api**
  API for automatic EarnApp device registration.
- **gcp-autostart**
  Real-time monitoring and auto-restart tool for GCP preemptible instances.
- **maccms-tool**
  Video information organizer for MacCMS.
- **mail-api**
  Email sending API with support for multiple templates.
- **mtab-import**
  Enrichment script for mTab new tab page (wallpapers, bookmarks).
- **oracle-lookbusy**
  Keep-alive tool for Oracle Cloud free instances, using intelligent resource occupation strategy.
- **wallpaper-dl**
  Multi-platform wallpaper downloader supporting Wallhaven, 360 Wallpaper.
